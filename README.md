# Project 1: Exploratory Data Analysis & Inference

## Dataset
NYC Yellow Taxi Trip Data (Kaggle)

## Project Objective
This project analyzes real-world NYC taxi trip data using EDA and basic statistical inference.

## Structure

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Descriptive Statistics
5. Confidence Intervals
6. Hypothesis Testing

## How to Run

Open the Jupyter Notebook file in Google Colab and run all cells from top to bottom.

Dataset is loaded using kagglehub.
